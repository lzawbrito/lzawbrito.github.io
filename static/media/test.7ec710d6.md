### Headers
Try to use the triple hashtag headers (`###`) because `h1` and `h2` are the 
site title and article title respectively.

### Fonts, Stylings, etc. 
Here is _italicized text_, here is **bold text**, here is some `code`, and 
here is ~a strikethrough~. Here is a code block, too:

```
def hello_world():
	print('hello world')
```
### Blockquotes 
Blockquotes look like this:

> Hi, I'm a blockquote. Lorem ipsum lorem ipsum so on.
> Yada yada. What happens if we go on to the next line? I'm curious.

### Math 
Here's some $\LaTeX$! This is ([Stokes' Theorem](https://en.wikipedia.org/wiki/Stokes%27_theorem)):

$$
\oint \mathbf{F}\cdot d \mathbf{r} = \iint_S (\nabla \times \mathbf{F}) \cdot d \mathbf{a}
$$
